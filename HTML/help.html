How do we add a new behaviour entry?
Each behaviour is unique, and will need to be added to each patient seperately. The option to add behaviours can be found in Home -> Assign -> [Patient name] -> Add a behaviour. Give a name and brief description of the behaviour, and set the timeout interval between button presses.

What is a "Timeout Interval"?
For more accurate logging, an interval between button presses has been implemented. This means that when a user logs a behaviour by pressing the button, there is a period of wait time before the button can be pressed again.
